Absolute Center Using CSS

Dalam CSS, sejauh yang saya tahu (AFAIK), belum ada fitur khusus untuk mengatur alignment sebuah gambar. Dalam kegiatan layouting, salah satu yang dibutuhkan adalah fitur untuk align center secara horizontal maupun vertical.

Ada banyak teknik untuk mengatur align center secara horizontal. Begitu pula dengan teknik align center secara vertical. keseluruhan teknik tersebut memiliki kesamaan yaitu teknik yang dibuat dengan men-hack CSS. dimana, salah satu ciri khas dari CSS adalah kombinasi property tertentu akan menghasilkan hasil diluar dari sifat properti tersebut. Sebagai contoh, kombinasi property flat memungkinkan untuk membangun sistem grid berbasis CSS, contohnya 960 grid system.

Horizontal Centering

salah satu teknik horizontal centering untuk image adalah dengan menggunakan property text-align: center;. property ini bisa digunakan pada elemen HTML yang memiliki property display: block. sebagai ilustrasi, perhatikan kode berikut. kode tersebut mensimulasikan sebuah skenario dimana terdapat container dari sebuah gambar yang memiliki ukuran lebih besar dari gambar tersebut.
